# 🌤️ Sunnyside Landing Page

Projeto de uma landing page fictícia de uma agência criativa, desenvolvido como desafio de front-end. O objetivo é praticar a construção de interfaces modernas, responsivas e com boa experiência de usuário (UX).

🔗 Live Demo: https://sabrinaz8.github.io/sunnyside/  
📁 Repositório: https://github.com/SabrinaZ8/sunnyside

---

## ✨ Sobre o projeto

O Sunnyside Landing Page simula o site de uma agência de marketing e design. O foco principal é apresentar serviços, cases e depoimentos de forma visualmente atrativa e organizada.

O projeto foi desenvolvido com base em um desafio de front-end, priorizando fidelidade ao layout, responsividade e atenção aos detalhes visuais.

---

## 🎯 Objetivos

- Praticar desenvolvimento de interfaces responsivas  
- Melhorar habilidades em HTML e CSS
- Trabalhar com layout moderno e hierarquia visual  
- Reforçar conceitos de UX/UI design  

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- Flexbox  
- Design responsivo
